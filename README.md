# Desenvolvimento Web
O projeto ONG Aurora Social foi desenvolvido com o objetivo de aplicar os fundamentos de HTML5, CSS3 e JavaScript, criando uma plataforma web funcional e acessível voltada para organizações sem fins lucrativos.
A proposta é oferecer uma presença digital moderna e responsiva para ONGs que atuam nas áreas de educação, alimentação e tecnologia, permitindo a divulgação de projetos sociais, o engajamento de voluntários e o recebimento de doações de forma transparente.

Além de cumprir os requisitos técnicos da disciplina, o site busca demonstrar boas práticas de semântica, acessibilidade (WCAG 2.1), organização de pastas, validação W3C, e otimização de imagens, representando um modelo de projeto real aplicável em contextos sociais.
